## Step-by-Step: Sign Up & Create Server

### DigitalOcean (Recommended)

1. Go to [digitalocean.com](https://www.digitalocean.com/)
2. Click "Sign Up" → Use email or GitHub
3. Verify email
4. Add payment method (credit card or PayPal)
5. You may get $200 free credit!
6. Click "Create" → "Droplets"
7. Choose Ubuntu 22.04
8. Select $6/month plan
9. Choose password authentication
10. Click "Create Droplet"
11. Save the IP address!

### Oracle Cloud (Free)

1. Go to [oracle.com/cloud/free](https://www.oracle.com/cloud/free/)
2. Click "Start for free"
3. Create Oracle account
4. Enter credit card (for verification only, won't be charged)
5. Wait for account approval (usually instant)
6. Go to Console → Compute → Instances
7. Click "Create Instance"
8. Name: rct-trader
9. Image: Ubuntu 22.04
10. Shape: VM.Standard.E2.1.Micro (Always Free)
11. Add SSH key
12. Click "Create"

## Common Questions

**Q: Can I run this on my home computer?**
A: Yes, but it won't be 24/7 unless you leave it on. Cloud VPS is more reliable.

**Q: Do I need a domain name?**
A: No, you can use the IP address directly.

**Q: Can I upgrade later?**
A: Yes, all providers allow easy upgrades.

**Q: What if I exceed the free tier?**
A: You'll be charged. Set up billing alerts!

**Q: Can I run multiple bots?**
A: Yes, but you'll need more RAM (2GB+ recommended).
